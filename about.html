À propos de ce site

Ce site a été réalisé dans le cadre de la formation Programmation de sites web du CNAM PACA.

Une attention particulière a été portée à la sémantique et à l’accessibilité.

Les articles ont été rédigés pour l’occasion, directement en HTML, avec Pug et Gulp en guise de CMS.

La page Ressources est une collection d’articles et outils utiles pour apprendre et pratiquer le développement web.

Un framework CSS a été développé pour l’occasion avec Stylus. Le code source de BASICSS est disponible sur Github.

Pour les polices, ce site prend le parti de se reposer sur le System Font Stack.

Les illustrations ont été réalisées avec Sketch.

Le site est hébergé sur Github Pages, et l’ensemble du code est consultable sur Github.

Une attention toute particulière a été portée aux apostrophes.
